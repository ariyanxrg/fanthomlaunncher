## 📌 About

Fanthom Launcher is a sleek, web-based platform designed to help gamers launch and manage their favorite games from a single, intuitive interface. Built with modern web technologies, it provides a smooth and responsive experience across all devices.

> **Note:** Currently in active development. The frontend is complete and the game launcher application is coming soon!

---

## ✨ Features

- 🎮 **Game Library** - Browse and organize your game collection
- ⚡ **Quick Launch** - One-click game launching
- 📱 **Responsive Design** - Works on desktop, tablet, and mobile
- 🎨 **Modern UI** - Clean and intuitive interface
- 🔍 **Search & Filter** - Find your games instantly
- 🌙 **Dark/Light Mode** - Choose your preferred theme

---

## 🛠️ Tech Stack

- **HTML5** - Structure and semantics
- **CSS3** - Styling and animations
- **JavaScript (Vanilla)** - Interactivity and logic
- **Netlify** - Deployment and hosting
