## 📌 About

**Fanthom Launcher** is a modern, responsive landing page for a next-generation gaming platform. It showcases premium features, pricing plans, team members, live status, and more — all in a sleek glassmorphism design with dark/light mode support.

> This is a complete front-end project demonstrating professional UI/UX, modular code structure, and smooth interactions.

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| 🎮 **Hero Section** | Engaging intro with live player count |
| 💎 **Feature Grid** | 4 core benefits: Speed, Flexibility, Community, Multi-Platform |
| 💰 **Pricing Plans** | 3 tiers (Starter, Pro, Ultimate) with tags |
| 👥 **Team Showcase** | Developer profiles with social links |
| 📊 **Live Status** | Service uptime, regions, and incident log |
| ❓ **FAQ Section** | Collapsible Q&A for user support |
| 🐛 **Bug Report** | Functional form for issue submission |
| 🌓 **Dark/Light Mode** | Seamless theme switching |
| 📱 **Fully Responsive** | Optimized for all screen sizes |
| 🔒 **Security** | Right-click & dev tools protection |
